# 一泽电竞

  应用介绍: "一泽电竞"app是一款购买VR产品(VR 眼镜、VR 书籍)的应用。

  应用功能: 
  
    商品分类：VR 眼镜、VR 书籍等类型商品。
    下单方法：直接选择单品，选择收件地址进行结算、加入购物车选择性结算。
    购买流程：应用内下单，商家免费配送，货到付款。
    签到兑好礼：订单流程结束获得对应数量积分，积分可直接在"签到兑好礼"兑换商品，同时签到也能获得积分。
    我的订单：商品下单的订单和积分兑换的商品，均在我的订单列表。
    设置：通用设置、收货地址以及关于VR。
